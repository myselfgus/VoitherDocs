## Conceitos Fundamentais

### Voither Dual Nature
**Voither** tem dupla natureza:
1. **Startup** - Empresa de healthtech inovadora
2. **Plataforma** - Private Edge Cloud-Mesh em rizomas

### Tecnologias Core
- **Cloudflare** - CDN global e segurança
- **Apple Silicon** - Processamento eficiente
- **Starlink** - Conectividade resiliente
- **LLMs Locais** - IA sem dependência externa

### Divisões Principais
- **Mestral Engine** - Motor de processamento inteligente
- **Sortio** - Saúde pública especializada
- **HealthOS** - Sistema operacional integrado
- **ResoAR** - Realidade aumentada médica
